Instructions
------------
1. The registry-entry.xml is an XML document, which probably created under a new RxT to store database connectivity parameters.
2. It is stored under the registry location '/_system/governance/applications/database/HRMDB/1.0.0/params' of the ESB.
3. The RegistryReadProxy.xml shows, how these values can be read from the xml file that is stored in the registry (governance root).
